## 2. 分段规则

### 2.1 公共

1. 保护注释块 `【…】` / `〈…〉`，注释内句号不切段。
2. **禁止**段首为 `【按…】` / `【臣】` / 残 `】`（并回上段）。
3. **例外**：纯 **`【干支】` 岁标**（`【甲午】`…）**必须另起段**；禁止孤立 `【丙申】。` 再下段写年事 → 并成 `【丙申】收国二年…`。
4. **禁止** AI 猜补收尾引号；仅允许删除 `云"短"续` 类误收短闭引（只匹配「云」）。
5. auto-punct 常「一句一行」→ 按规则**组段**，不是每行一段。

### 2.2 `chrono`（编年 / 会编 / 要录 / 大金国志）

可断段首：

- 年号年：`天会四年` / `建炎元年` / `绍兴十年`…
- 季节（可带月）：`春正月` / `夏` / `秋` / `冬`
- 相对年：`是岁` / `是年` / `次年` / `明年` / `翌年`（含 `次年三月`）
- 干支日、引书句式
- **`【干支】` 岁标**
- **句中时间 peel**（`peel_inline_time_breaks`）：在 `。！？；` / `】` 后若接月份、季节起事、是岁、年号年、`【干支】` → 强制另起  
  - **保护**：不切开 `【丙申】收国二年…`

### 2.3 `kaoju`（七国考等）

| 要 | 不要 |
|----|------|
| 门类、词条单独段 | 年号/季节/干支断（那是 chrono） |
| `策：`（战国策简写）、`史记：` / `X传：` / `通鉴：` | 把 `相国策` 整吞为 `国策` |
| `按` / `余按` / `又…` 起段 | 按礼内职级罗列切成伪词条 |

开卷权威结构示例：

```text
钦定四库全书七国攷卷一明董说撰。
秦职官〈封君后妃附〉
相国
策：卫鞅亡，入秦孝公以为相。
史记：…乐池相秦。
按礼：…
```

### 2.4 四库卷次（大金国志）

每卷三层，**各自单独段**：

1. 卷名：`钦定四库全书钦定重订大金国志卷一`
2. 概括：`纪年太祖武元皇帝上` / `开国功臣传` / `文学翰苑上`…
3. 正文：起传、年事；岁次 `【辛丑】天辅五年春…`

另：开卷目录每行一卷；提要整段；删卷尾重复卷标；**卷号长优先**（禁「卷三十七」→「卷三」）。

全本门控：plain 汉字量约 **≥14–15 万**、卷首～卷四十一，才可当全本入库（~135KB 级为残本）。

### 2.5 🔴 卷首《金九帝年谱》特殊

**年谱 ≠ 正文编年。**

- 每位皇帝 **一整段**（从 `太祖武元皇帝【` 到下一帝号前）
- **禁止** 用正文时间 peel / soft_wrap 按月、改元拆碎
- 实现：`split_dajin_juan_shou_body` + `merge_dajin_nianpu_emperor_blocks`；peel 在「金九帝年谱」→「卷一」整区跳过
- 帝号：`海陵，炀王` → **`海陵炀王`**；`海陵，炀主` → **`海陵炀主`**（**只去逗号，不改主/王**）
- 验收：标题 + 9 帝 + `金九帝起` ≈ 11 段；orphan 月首 = 0  
- **任何**改写大金国志的流水线结束必须抽查卷首年谱

## 3. 断句规则（句内，`repair_false_punct`）

> **断句 ≠ 分段。**  
> - **分段**（`resegment`）：段与段之间何时另起（年号、季节、卷名、年谱帝条…）。  
> - **断句**（`repair`）：**同一段/同一句内部**，模型把专名、官制、年月从中间切开后如何回粘，以及少量高置信语义逗号如何补回。  
> 用户说「断句差」时，**先查句内误断**，不要只会讲分段规则。

### 3.1 问题从哪来

自动标点模型（`guwen-biaodian`）常见病：

| 类型 | 烂例 | 说明 |
|------|------|------|
| 地名切开 | `东。平` `云。中` `紫。宸` `淮。汉` | 二字/多字地名中间插 `。`/`，` |
| 官署切开 | `元帅，府` `枢密院。事` `州。郡` | 官制专名 |
| 年月切开 | `天会四，年` `六，月` `是年六。月` | 年号+年、数+月 |
| 人名/帝号切开 | `高。永昌` `海陵，炀王` | **人名中间不断** |
| 动宾/成语切开 | `父，忧` `侵，淮` `南，伐` | 习惯连写词 |
| 合法句界被误粘 | （若全局删句号）`死领` `东平元帅府` 丢逗号 | **禁止用 raw 全局对齐** |

模型评级（实测）：**B- / 可检索浏览，不是精校定本**。引号乱套、并列职级用句号等，repair **管不全**。

### 3.2 铁律（用户定稿，必须遵守）

1. **不要重跑 guwen-biaodian 指望专名变好** — 同样会切；用规则 + 无标点 raw。  
2. **禁止「凡 raw 二字连写就删中间 `。`」的全局对齐** — raw 换行处本无标点，会吃掉合法句界（曾导致 `死领`、丢 `归至东平，元帅府` 的逗号）。  
3. **只删/并「专名与年月内部」的误断**；语义逗号单独、高置信补。  
4. **词表目标字用简体**（先 `normalize_variants`）。勿只留 `枢宻`/`髙` 作唯一键，否则洗完异体后 repair 0 命中。  
5. **顺序：`resegment` 之后再 `repair`**。repair 不负责卷层/干支岁标/年谱结构。  
6. **用户点名一个误断 → 立刻加词表/HARD_FIXES**，不要等「以后统一精校」。  
7. **只去误断标点，不擅自改字**（见 3.3）。  
8. **规则修复 ≠ 全书精校** — 可继续贴烂段加规则；勿谎称完成。  
9. **不补引号**（与分段铁律一致）：不猜位置补 `"`；只允许删 `云"短"续` 类误收短闭引。

### 3.3 只去标点、不改字（用户怒改）

| 底本形态 | 正确 | 错误 |
|----------|------|------|
| `海陵，炀王` | `海陵炀王`（去逗号） | — |
| `海陵，炀主` | **`海陵炀主`**（只去逗号） | ~~改成海陵炀王~~（擅自改「主」→「王」） |
| `高。永昌` | `高永昌` | 删字、换字 |

**原则：** 专名内部的 `，。、` 可去；**字用哪个以底本/用户点名为准**，不能因为「别处都写王」就统一替换。

### 3.4 实现分层（`repair_false_punct.py`）

按顺序：

```text
HARD_FIXES（精确字符串）
  → COMPOUNDS 词表回粘（字间 [，。、；：]*）
  → raw 词库回粘（仅 raw 中确有的整词：州府军县京… + 白名单）
  → unsplit_dates（年号年 / 数月 / 是年+季月）
  → semantic_commas（高置信语义读）
  → repair_inside_annots（【按…】内部同样跑）
  → 末轮再 semantic_commas + unsplit_dates
```

| 层 | 作用 | 例 |
|----|------|----|
| **HARD_FIXES** | 用户点名、易混、不能靠通用规则 | `海陵，炀主`→`海陵炀主`；`高。永昌`→`高永昌` |
| **COMPOUNDS** | 地名/官制/交涉专名/人名等；长词优先 | `东平` `元帅府` `枢密院事` `燕山地` `石晋故地` `高永昌` `海陵炀王` |
| **raw 词库** | 从无标点底本抽 `X州/府/军…`；**仅 raw 连写的整词** | 禁跨词假专名如「平元帅府」 |
| **日期** | `天会四，年`→`天会四年`；`六，月`→`六月`；`是年六。月`→`是年六月` | 不动「月，宋」类合法句读 |
| **semantic_commas** | 回粘后补**该有的**逗号/句号 | 见下 |

### 3.5 词表回粘（COMPOUNDS / unsplit_word）

- 对词 `W = c1 c2 … cn`：匹配 `c1` + 可选标点 + `c2` + … → 整词 `W`。  
- **长词优先**（`sorted(..., key=len, reverse=True)`），避免短词抢先。  
- 覆盖类别（实现中持续追加，以下为已定类，非穷尽）：  
  - **地名**：东平、云中、燕京、西京、河朔、燕云、磁州、单州、济南、居庸关、天祚…  
  - **官署/官制**：元帅府、枢密院、枢密院事、签书院事、万户、都统、父忧、起复、州郡、知东平…  
  - **交涉专名**：燕山地、石晋故地、平营滦、赵良嗣、马扩、岁币、夹攻…  
  - **帝号/人名**：海陵炀王、高永昌（用户点名追加）  
- 用户新点名 → **同时**加 COMPOUNDS + 必要时 HARD_FIXES。

### 3.6 日期误断（unsplit_dates）

| 烂 | 应 |
|----|----|
| `天会四，年` / `天辅七。年` | `天会四年` / `天辅七年` |
| `六，月` / `十。月` / `十一，月` | `六月` / `十月` / `十一月` |
| `是年六。月` / `是岁，春` | `是年六月` / `是岁春`（模式内） |

约束：只合「数+月」「年号+数年」；**不要**把「六月，宋遣…」的逗号当误断删掉（那是合法句读，归分段/叙事）。

### 3.7 语义逗号 / 句读（semantic_commas）

词表回粘后，有些地方**需要逗号或句号**才符合阅读与用户锚句：

**用户权威锚句（大金国志，修完必须贴近）：**

```text
四月，尼雅满自…归至东平，元帅府差刘豫知东平府兼节制河南诸州郡，豫先为宋济南守降于我，至此除知东平。六月，行下削发及禁民汉服，不如式者死。领燕京枢密院事刘彦宗卒，并枢密院于云中，除云中韩企先为相，同时立爱主之。…侵淮，从之。【按…是年六月…取磁单等州…】
```

高置信规则类（脚本内维护，摘要）：

| 模式 | 目标 |
|------|------|
| `归至东平元帅府` | `归至东平，元帅府` |
| `诸州郡豫先` / `降于我至此` | `诸州郡，豫先` / `降于我，至此` |
| `知东平六月` | `知东平。六月` |
| `六月行下` | `六月，行下` |
| `不如式者死领` | `不如式者死。领` |
| `为相同时` | `为相，同时` |
| `侵淮从之` | `侵淮，从之` |
| `博索渤海万户` | `博索、渤海万户` |
| `以彦宗之故命` | `以彦宗之故，命` |
| 燕山地 / 石晋故地 / 蓟景檀顺… | 交涉串专用补读 |

另有双保险：`云。中`→`云中`、`东。平`→`东平`、`元帅，府`→`元帅府`、`天。祚`→`天祚` 等写在 semantic 层，与 COMPOUNDS 互补。

### 3.8 注释内同样修

`【按…】` 里也会出现 `天会四，年` / `父，忧` / `云。中`。  
`repair_inside_annots`：对注释块内部再跑词表/日期/语义，**不要**只修正文不修按语。

### 3.10 与分段的交界（别混）

| 现象 | 归谁 |
|------|------|
| `…事。十一月，议割燕山…` 整段过长 | **分段** `peel_inline_time_breaks` |
| `十一月` 被写成 `十一，月` | **断句** `unsplit_dates` |
| 九帝年谱被拆成「五月乙丑」「十二月」多段 | **分段** 年谱特殊规则 |
| `海陵，炀王` 帝号中间逗号 | **断句** HARD_FIXES / compounds |
| `【丙申】。` 与下年事两段 | **分段** `merge_lonely_gz_year_marks` |

### 3.11 验收（断句专项）

```text
□ 东。平 / 云。中 / 天。祚 == 0
□ 数，月 / 年号数，年 误断 == 0（日期层后）
□ 锚句：归至东平，元帅府；知东平。六月；侵淮，从之；取磁单等州
□ 海陵，炀王 / 海陵，炀主 误断 == 0；且「炀主」处仍是「主」不是被改成「王」
□ 高。永昌 == 0
□ 未做全局 raw 对齐删句号
□ 不声称全书精校完成
```

### 3.12 操作 SOP（用户再丢烂句时）

1. 定位活跃文件中的误断上下文。  
2. 查无标点 raw 是否连写（人名/地名）。  
3. 加 **COMPOUNDS** 和/或 **HARD_FIXES**（只去标点、不改字）。  
4. 若缺语义逗号 → 加 `semantic_commas` 规则（写清前后文，避免过宽）。  
5. 跑 `repair_false_punct.py`（或局部 replace）→ 抽查锚句 + 用户新点名。  
6. 同步更新本文件 + skill 的「用户已点名清单」。
